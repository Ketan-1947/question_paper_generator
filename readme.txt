This is Question paper Generator.
This is tool used to generate question paper in text format by following a certain pattern of three sections.

Made using following:
                    Python:
                        1.Tkinter : for user interface
                        2.mysql.connector : for python and mysql connectivity
                        3.Random : for randomizing question picking
                        4.PIL : for background images
                        5.messagebox : to show warnings, info and errors


                    MySQL:
                        1.Used to store all questions in different tables named after subjects.
                        2.Each table contains two colums first is question and second is marks corresponding to the question
